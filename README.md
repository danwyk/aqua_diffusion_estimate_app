## 水环境污染物扩散模型 - Demo

### Backgroud
这是一个水环境污染物扩散的演示模型，用于演示   
* 可溶性污染物   
* 油类污染物   
在 “流动” 或者 “静止” 的河道中的扩散情况   

---

### Implementation
Vue2, Kriging, OpenLayers, Echart, Element-UI, Electron, Nodejs
